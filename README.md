# datawalker-site

Official DataWalker website – AI Development &amp; Consulting

Список команд:

- exit — завершує сеанс терміналу
- pwd — показує поточне місцезнаходження у файловій структурі
- ls — виводить список файлів і папок у поточній робочій директорії
- cd .. — переміщує назад на один рівень (вкладеність, папку), тобто до попередньої директорії
- cd [ім'я директорії] — переміщує вперед, тобто до вкладеної директорії
- cd ~ — повертає в кореневу директорію
- clear — очищує вміст терміналу

Збережи зміни в локальному та віддаленому репозиторії

- Використай команду git status, щоб перевірити статус локального репозиторію
- Введи команду git add --all , щоб додати всі зміни під контроль Git
- Повтори команду git status, щоб переконатися, що зміни під контролем
- Введи команду git commit -m “add index file”, щоб зберегти зміни в історії
- Введи команду git push, щоб відправити зміни на GitHub
