1. Модуль 1. Основи HTML

Програма модулю

Цілі Модуля 1:

  • зрозуміти, як працює вебсторінка
  • навчитися використовувати основні теги та їх атрибути
  • ознайомитися з принципами семантичної розмітки та застосовувати їх на практиці
  • дізнатися, що таке структурна розмітка і як вона співвідноситься зі структурою HTML-документа
  • визначити алгоритм створення веб-сторінки
  • розібратися з форматами зображень та навчитися переносити їх із макета в проєкт.

Веб-технології

Веб-технології HTML, CSS, JS

Вебсайти – це основа, на якій тримається інтернет. І всі вони побудовані на трьох основних технологіях.

  • HTML (HyperText Markup Language) – мова гіпертекстової розмітки. Вона відповідає за структуру сторінки: заголовки, абзаци, списки, посилання, зображення тощо.
  • CSS (Cascading Style Sheets) – каскадні таблиці стилів. Вони відповідають за зовнішній вигляд сторінки: кольори, шрифти, відступи, розташування елементів тощо.
  • JavaScript (JS) – мова програмування. Вона відповідає за інтерактивність сторінки: анімації, обробку подій, динамічне завантаження контенту тощо.

Інструменти розробника

Розробники можуть подивитися, як влаштовані сайти, на які вони заходять.

Для цього в браузерах є вбудовані інструменти розробника (DevTools).

Вони дозволяють:

  • переглядати та редагувати HTML і CSS на льоту;
  • відлагоджувати JavaScript-код;
  • аналізувати продуктивність сторінки;
  • перевіряти адаптивність дизайну;
  • переглядати мережеві запити;
  • та багато іншого.

Щоб відкрити інструменти розробника, можна:

  • - натиснути клавішу F12 або скористатися контекстним меню (правий клік → "Перевірити" або "Inspect").
  • - комбінацією клавіш Ctrl+Shift+I для Windows
  • - комбінацією клавіш Cmd+Opt+I для MacOS

HTML-документ - текстовий файл з розширенням .html.

HTML-документ складається з елементів, які визначають структуру та зміст веб-сторінки.

Кожен елемент має відкриваючий та закриваючий теги, між якими розміщується вміст елемента.

Наприклад, елемент заголовка виглядає так:

<h1>Це заголовок</h1>

Тут <h1> - відкриваючий тег, Це заголовок - вміст елемента, </h1> - закриваючий тег.

HTML-документ має певну структуру, яка включає в себе такі основні частини:

  • <!DOCTYPE html> - оголошення типу документа;
  • <html> - кореневий елемент документа;
  • <head> - метаінформація про документ;
  • <body> - основний вміст документа.

Ось приклад простої структури HTML-документа:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Назва сторінки</title>
</head>
<body>
    <h1>Привіт, світ!</h1>
    <p>Це мій перший HTML-документ.</p>
</body>
</html>

У цьому прикладі ми бачимо всі основні частини HTML-документа.

Тепер розглянемо деякі основні теги HTML та їх атрибути.

Основні теги HTML

  • <h1> - <h6> - заголовки різних рівнів;
  • <p> - абзац тексту;
  • <a> - гіперпосилання;
  • <img> - зображення;
  • <ul> - ненумерований список;
  • <ol> - нумерований список;
  • <li> - елемент списку;
  • <div> - блоковий елемент;
  • <span> - рядковий елемент;

Атрибути HTML

Атрибути надають додаткову інформацію про елементи HTML.

  • href - використовується з тегом <a> для вказівки URL посилання;
  • src - використовується з тегом <img> для вказівки шляху до зображення;
  • alt - використовується з тегом <img> для надання альтернативного тексту;
  • title - надає додаткову інформацію про елемент;
  • class - використовується для визначення класу елемента для CSS стилів;
  • id - використовується для унікальної ідентифікації елемента;
